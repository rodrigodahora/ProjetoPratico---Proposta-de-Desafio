# ProjetoPratico - Desafio de Carreira M03

<p>Este projeto é focado na criação de um sistema de gerenciamento de formulários que inclui as operações básicas do CRUD - create, read, update and Delete de dados. 
Basicamente uma iniciativa para além da Cubos Academy, minha formação em Desenvolvimento de Software em andamento, tal iniciativa parte da crença na necessidade de estudos incessantes para o aperfeiçoamento de técnicas e da memória, que permite a um programador ou programadora
se recordar mais facilmente da usuabilidade de códigos já utilizados. Em resumo, exercitar é tão eficaz quanto o estudo teórico, por esse motivo é interessante a dedicação em projetos suplementares.
</p>

<img src="https://bnz06pap001files.storage.live.com/y4mKVuq8WJ1qw4c1x_SCHApUtczHwdR3xEYGY-AOsmBm_CdexfVxRUrHgErctvpH4qDTL1406gHtH-SkEpi8Zs7Z70K4OPAicDtefCBIEN65z_-TIyet_8IA2RMwyIaCaC_LkG_oNAaaWWYv0O5ftYorRQ81re3wra3tr9ymxnmqzeS6xx5TmB3YR_jBHrfF3k4?width=1365&height=668&cropmode=none" alt="Pagina de Cadastro" height="400" width="800" src="">
  <p>A página de cadastro requisita do usuário o preenchimento supracitado do formulário para que seja feito o submit dos dados para armazenamento</p>
  <p>Na página de cadastro também é terminantemente necessário o preenchimento dos campos de input para que seja dada a continuidade na submição dos dados preenchidos. O botão "Cadastrar" faz o submite e redireciona o usuário para a página inicial e assim permite que seja feito o login com a conta cadastrada.</p>
  <p>Se já existir conta cadastrada o usuário poderá utilizar o botão Cancelar ou até mesmo o link abaixo "Já tem cadastro" para que possa retroceder à área de Login.</p>
</div>
